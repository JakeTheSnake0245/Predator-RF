# Predator SDR

A passion project aimed at lowering the barrier to entry for working in the electromagnetic environment (EME). Getting started with SDR has a steep on-ramp — understanding the tools, the spectrum, and how to navigate it all takes time. Predator SDR is an attempt to make that easier.

## About

Predator SDR is built on top of [SDR++](https://github.com/AlexandreRouma/SDRPlusPlus) and incorporates several other open-source projects. Most of the underlying tooling is carried over and re-skinned; the goal is a cleaner, more mission-focused experience rather than a from-scratch rewrite.
